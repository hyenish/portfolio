TVING

Viewport: web / tablet / mobile
Palette:

기간: 2022.12.
React로 axios를 이용하여 TMDB사이트에서 영화데이터를 API로 받아와 티빙컨셉의 OTT사이트를 만들었습니다. 또한 컨셉을 티빙으로 잡았기 때문에 한국 OTT 사이트의 느낌을 내기 위해 데이터의 언어를 한글버전으로 받아왔습니다.

메인 페이지는 API 데이터에서 인덱스 값으로 하나의 작품을 불러와 화면의 메인 이미지로 보이게 배치하였고, multi carousel을 이용하여 실시간 인기영화 / 평점 높은 인생영화 / TVING SOON으로 나눠 현재 개봉작과 평점 높은 작품 그리고 개봉 예정작을 슬라이드하여 넘겨볼 수 있도록 하였습니다.

서브 페이지는 홈화면에서의 작품을 누르면 해당 작품의 장르, 평점, 관객수 그리고 줄거리와 메인 예고편을 볼 수 있는 버튼을 만들었습니다. 몇 작품은 한글버전의 줄거리와 예고편이 제공되지 않아서 그런 작품은 줄거리 대신 "한국 버전에서는 제공하지 않는 작품입니다."라는 문장이 나타나게 하였습니다.

메종키츠네

View Port: web / tablet / mobile
Palette :

기간: 2022.12.
기존 메종키츠네 쇼핑몰 사이트에서 상품이미지와 제품명, 사이즈 등의 데이터를 가져와 직접 json파일을 만들었고, json파일을 로컬이 아닌 곳에서도 불러 올 수 있도록 My Json Server 사이트를 통해 서버와 연결하도록 하였습니다. 또한 React Router를 이용하여 영역별로 분할하여 만든 쇼핑몰 사이트 입니다.

제품 이미지인 card component를 클릭했을 때 로그인이 된 상태여야 상세페이지를 볼 수 있도록 로그인 유무에 따라 로그아웃 상태에서는 로그인 화면으로, 로그인 상태에서는 상세페이지로 넘어가도록 하였습니다.
